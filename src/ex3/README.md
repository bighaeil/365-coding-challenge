# Day 001 - 유효한 괄호 (Valid Parentheses)

## 문제

문자열 `s`가 `(`, `)`, `{`, `}`, `[`, `]` 로만 이루어져 있을 때, 괄호가 올바르게 짝지어져 있는지 판별하세요.

## 조건

1. 열린 괄호는 같은 종류의 닫힌 괄호로 닫혀야 한다.
2. 열린 괄호는 올바른 순서로 닫혀야 한다.
3. 모든 닫힌 괄호에는 대응하는 열린 괄호가 있어야 한다.

## 예시

| 입력 | 출력 | 설명 |
|------|------|------|
| `"()"` | `true` | 올바른 짝 |
| `"()[]{}"` | `true` | 여러 종류 올바른 짝 |
| `"(]"` | `false` | 종류가 다름 |
| `"([)]"` | `false` | 순서가 잘못됨 |
| `"{[]}"` | `true` | 중첩된 올바른 짝 |
| `""` | `true` | 빈 문자열은 유효 |

## 힌트

- 스택(Stack) 자료구조를 활용해보세요.
- 시간복잡도: O(n), 공간복잡도: O(n)

## 풀이

### 접근 방식: 스택(Stack) 활용

1. 열린 괄호(`(`, `{`, `[`)를 만나면 스택에 push
2. 닫힌 괄호(`)`, `}`, `]`)를 만나면 스택의 top과 짝이 맞는지 확인
3. 짝이 맞으면 pop, 아니면 `false` 반환
4. 순회가 끝난 후 스택이 비어있으면 `true`

### 코드

```javascript
function isValid(s) {
  const stack = [];
  const pairs = {
    ")": "(",
    "}": "{",
    "]": "[",
  };

  for (const char of s) {
    if (char === "(" || char === "{" || char === "[") {
      stack.push(char);
    } else {
      if (stack.length === 0 || stack[stack.length - 1] !== pairs[char]) {
        return false;
      }
      stack.pop();
    }
  }

  return stack.length === 0;
}
```

### 복잡도

- 시간복잡도: **O(n)** — 문자열을 한 번 순회
- 공간복잡도: **O(n)** — 최악의 경우 모든 문자가 열린 괄호
