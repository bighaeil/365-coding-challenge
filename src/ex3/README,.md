# Day 001 - 유효한 괄호 (Valid Parentheses)

## 문제

문자열 `s`가 `(`, `)`, `{`, `}`, `[`, `]` 로만 이루어져 있을 때, 괄호가 올바르게 짝지어져 있는지 판별하세요.

## 조건

1. 열린 괄호는 같은 종류의 닫힌 괄호로 닫혀야 한다.
2. 열린 괄호는 올바른 순서로 닫혀야 한다.
3. 모든 닫힌 괄호에는 대응하는 열린 괄호가 있어야 한다.

## 예시

| 입력 | 출력 | 설명 |
|------|------|------|
| `"()"` | `true` | 올바른 짝 |
| `"()[]{}"` | `true` | 여러 종류 올바른 짝 |
| `"(]"` | `false` | 종류가 다름 |
| `"([)]"` | `false` | 순서가 잘못됨 |
| `"{[]}"` | `true` | 중첩된 올바른 짝 |
| `""` | `true` | 빈 문자열은 유효 |

## 힌트

- 시간복잡도: O(n), 공간복잡도: O(n)
