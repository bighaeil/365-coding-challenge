# ex19. 숫자 문자열과 영단어 (Number String and English Words)

## 문제 설명

네오와 프로도가 숫자놀이를 하고 있습니다.
네오가 프로도에게 숫자를 건넬 때 일부 숫자를 영단어로 바꾼 후 전달합니다.

각 숫자에 대응되는 영단어는 다음과 같습니다.

| 숫자 | 영단어 |
|------|--------|
| 0 | zero |
| 1 | one |
| 2 | two |
| 3 | three |
| 4 | four |
| 5 | five |
| 6 | six |
| 7 | seven |
| 8 | eight |
| 9 | nine |

문자열 `s`가 주어질 때, 이를 **숫자로 변환한 결과**를 반환하세요.

> 카카오 2021 신입 공채 코딩테스트 기출 문제입니다.

## 입력 조건

- `s`의 길이는 `1` 이상 `50` 이하입니다.
- `s`가 나타내는 숫자는 `1` 이상 `2,000,000,000` 이하입니다.
- 문자열 `s`는 숫자와 영단어가 섞여 있습니다.

## 예제

**예제 1**
```
입력: s = 'one4seveneight'
출력: 1478
```

**예제 2**
```
입력: s = 'twentyfour' (X, 두 자리 숫자는 없음)
입력: s = '23four5six7'
출력: 234567
```

**예제 3**
```
입력: s = '2three45sixseven'
출력: 2345607  (X)
출력: 2345 67  (X)
출력: 23456 7  (X)

입력: s = '2three45sixseven'
출력: 2345 → 아니고 각 자리 수: 2,3,4,5,6,7 → 234567
```

**예제 3 (수정)**
```
입력: s = 'one1two'
출력: 112
```

## 입력 조건 정리
- 숫자와 영단어가 섞여 있으며 각 영단어는 0~9 한 자리 숫자를 의미합니다.

## 예제 (최종)

**예제 1**
```
입력: s = 'one4seveneight'
출력: 1478
설명: one=1, 4, seven=7, eight=8 → 1478
```

**예제 2**
```
입력: s = '23four5six7'
출력: 234567
설명: 2, 3, four=4, 5, six=6, 7 → 234567
```

**예제 3**
```
입력: s = 'one1two'
출력: 112
설명: one=1, 1, two=2 → 112
```

## 풀이 아이디어

### 방법 1. 순차 replace — O(n)
영단어 목록을 순서대로 돌며 `replaceAll`로 숫자로 교체 후 `Number` 변환

```javascript
'one4seveneight'
→ replace 'zero'→'0': 'one4seveneight'
→ replace 'one' →'1': '14seveneight'
→ replace 'seven'→'7': '14eighteight' (X, seven이 포함된 경우 주의)
→ ...
```

### 방법 2. 순차 탐색 — O(n × m)
문자열을 앞에서부터 순회하며 현재 위치에서 영단어가 시작하는지 확인

## 면접 팁

- **"replaceAll을 쓰면 왜 안 되는 경우가 있나요?"** → `'eightwo'` 같이 영단어가 겹치는 경우 순서에 따라 결과가 달라질 수 있습니다. 앞에서부터 순차 탐색이 더 안전합니다.
- **"정규식으로도 풀 수 있나요?"** → 네. `s.replace(/zero|one|two|.../g, match => words.indexOf(match))` 로 한 번에 처리 가능합니다.
- **카카오 실전 팁**: 문자열 구현 문제는 **엣지 케이스(영단어 중첩, 숫자만 있는 경우, 영단어만 있는 경우)** 를 꼭 확인하세요.

### 🔍 동작 흐름 (`'one4seveneight'`)

```
초기:    'one4seveneight'
zero→0:  'one4seveneight'  (변화 없음)
one→1:   '14seveneight'
two→2:   '14seveneight'    (변화 없음)
...
seven→7: '14seven8'  → 아니고
seven→7: '147eight'
eight→8: '1478'

Number('1478') → 1478 ✓
```

### 💡 `'eightwo'` 엣지 케이스는?

```
eight→8 먼저: '8wo'  (eight가 two보다 먼저 처리됨)
two→2:        '82'
→ 82 ✓
```

`WORDS` 배열이 `0~9` 순서라 `eight(8)`가 `two(2)`보다 먼저 처리되어 자연스럽게 해결됩니다.
