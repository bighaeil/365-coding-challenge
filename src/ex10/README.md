# ex10. 클라이맍킹 계단 (Climbing Stairs)

## 문제 설명
n개의 계단을 오를 때, 한 번에 1계단 또는 2계단씩 오를 수 있습니다.
n개의 계단을 오르는 방법의 수를 반환하세요.

## 입력 조건
- 1 ≤ n ≤ 45

## 예제
입력: n = 2
출력: 2
설명: (1+1), (2)

입력: n = 3
출력: 3
설명: (1+1+1), (1+2), (2+1)

입력: n = 5
출력: 8

## 풀이 아이디어
- 피보나치 수열과 동일한 패턴
- n번째 계단 = (n-1)번째 경우의 수 + (n-2)번째 경우의 수
- 반복문으로 O(n), 공간 O(1) 풀이 가능

# 

물론이죠! ex9 방식 그대로 설명해드릴게요.

---

**초기 상태**

```
prev = 1   (n=1 경우의 수: 1가지)
curr = 2   (n=2 경우의 수: 2가지)
```

---

**n=5 일 때 반복 과정**

```
i=3:  next = 1 + 2 = 3  →  prev=2,  curr=3
i=4:  next = 2 + 3 = 5  →  prev=3,  curr=5
i=5:  next = 3 + 5 = 8  →  prev=5,  curr=8
```

결과: `curr = 8` ✅

---

**왜 prev + curr 인가요?**

n번째 계단에 도달하는 방법은 딱 두 가지뿐이에요.

```
n-1번째 계단에서 1칸 오르기  →  prev 가지
n-2번째 계단에서 2칸 오르기  →  그 전 prev 가지
```

그래서 항상 **이전 두 값을 더하면** 현재 경우의 수가 나와요.

---

**포인터 이동 시각화**

```
1계단  2계단  3계단  4계단  5계단
  1      2      3      5      8
  ↑      ↑
prev   curr

         ↑      ↑
       prev   curr

                ↑      ↑
              prev   curr

                       ↑      ↑
                     prev   curr  ← 최종 답
```

매 단계마다 두 포인터가 오른쪽으로 한 칸씩 밀리는 구조예요 😊
