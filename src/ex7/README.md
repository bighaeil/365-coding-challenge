# ex7. 숫자 배열에서 가장 큰 수 만들기

## 문제 설명
정수 배열 `nums`가 주어질 때, 배열의 숫자들을 이어 붙여 만들 수 있는
가장 큰 수를 문자열로 반환하세요.

## 입력 조건
- 1 ≤ nums.length ≤ 100
- 0 ≤ nums[i] ≤ 10,000

## 예제
입력: nums = [10, 2]
출력: "210"

입력: nums = [3, 30, 34, 5, 9]
출력: "9534330"

입력: nums = [0, 0]
출력: "0"

## 풀이 아이디어
- 두 수 a, b를 비교할 때 "ab" vs "ba" 문자열 비교로 정렬
- 정렬 후 이어 붙이기
- 모두 0인 경우 "0" 반환
